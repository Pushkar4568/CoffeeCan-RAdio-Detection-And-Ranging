# CoffeeCan-RAdio-Detection-And-Ranging
MIT Cantenna prototype
2.4 GHz FMCW radar system based on the MIT Cantenna Radar design. Includes RF front-end (VCO, PA, LNA, mixer), analog baseband processing, NE555 sweep generator, and Python/MATLAB signal processing for range detection. Final-year engineering project.
